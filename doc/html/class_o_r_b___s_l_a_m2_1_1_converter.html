Detailed Description

Convertidor entre g2o y opencv.

Clase no instanciada, sin propiedades, conjunto de métodos de clase. Convierte elementos entre las formas de expresión de opencv y g2o. Convierte en ambas direcciones Point, Vector, Mat, Egien::Matrix, g2o::sim3, g2o::SE2Quat.
